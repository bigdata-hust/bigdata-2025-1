About This Model

Sentiment Analysis Model

Mô hình phân tích cảm xúc được xây dựng bằng Apache Spark ML Pipeline, phục vụ bài toán Sentiment Analysis trên tập dữ liệu review.

Pipeline xử lý và huấn luyện bao gồm các bước:

  • Tokenizer – tách văn bản thành các token
  • StopWordsRemover – loại bỏ stop words
  • CountVectorizer – chuyển văn bản sang vector
  • IDF – tính trọng số TF-IDF
  • Logistic Regression – mô hình phân loại cảm xúc

Model được huấn luyện , sau đó save/load bằng PipelineModel và tích hợp vào Flask Web Application để dự đoán realtime.

Hệ thống phù hợp cho các bài toán Big Data NLP .
